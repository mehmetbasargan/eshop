## Kullandığı Teknolojıler

Aşağıdaki Teknolojileri kullanarak geliştirdiğim mini shop uygulaması.

React - Next.js 13+ (Typescript),
Tailwindcss,
React Slick Slider,
Framer Motion,
React Redux

## Başlarken

Başlarken:

```bash
npm install
#or
yarn

npm run dev
# or
yarn dev
```

Sonucu görmek için http://localhost:3000 adresini tarayıcınızla açın.
